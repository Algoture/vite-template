# A ShadCN + Vite + React + JavaScript template

A Vite.js template for building apps with ShadCN UI and JavaScript.

## Usage

```bash
git clone https://github.com/Algoture/vite-template
```

## Features

- Radix UI Primitives
- Tailwind v3.4.17
- ShadCN UI with JS
- Icons from [Lucide](https://lucide.dev)
